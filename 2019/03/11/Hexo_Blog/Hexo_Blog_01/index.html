【配置博客】01Windows操作系统搭建博客

发表于 2019-03-11 | 更新于 2019-03-19 | 评论数： | 阅读量：

搭建及部署博客。其中的过程很麻烦，但是按照步骤一步一步的做，其实很容易就能做到。我的系统是Win10,64位的操作系统。在我搭建博客的时候，看的一个MacOS操作系统搭建的博客。其中虽然差别不大，但是也让我走了很多弯路。如果过程顺利，最多半个小时就可以搭建好博客。以下的过程是在Win10操作系统下进行的，如果大家是Windows操作系统，可以参考以下步骤搭建博客。

前言


搭建博客框架最好一次完成，因为其中可能会遇到很多问题。遇到问题是最好一次就解决了，不要丢到下一次。我之前看大佬的视频安装，他们一步一步做的都很顺利，我跟着他们一步一步的做，却遇到了很多的问题。要么百度，要么卸载软件重新安装，这个过程很折磨人。但是学会以后，应该可以在二十分钟之内就搭建好一个博客的框架了。我在第一次搭建博客框架花了一天的时间，遇到了各种各样的问题，而且还解决了。建好之后还是很开心的！所以，搭建博客这种事情，尽量一次就做完，不要分几次才做好。

在这里，我是用的是Hexo的博客框架。因为我只用过这一种博客框架，所以也不能评价这个博客框架与其他博客框架的优劣，不过这一款博客的框架使用其实很方便的。Hexo的博客的官网，有几个醒目的大字快速、简介而高效的博客框架，应该也不会平白无故的吹嘘自己。更具体的内容，我在以后的博客中会写到。

准备工作

首先，我创建了一个文件夹，将所需要的文件全部放到了同一个文件夹中。
我个人认为，这样比较方便实用。但事实情况下，好像并不是这样。

1.注册Github账号

Github的官方网站 (https://github.com) ，注册账号登录。

2.创建一个仓库(repository)

点击如下按钮Your repositories
在跳转到的页面中，点击一个绿色的new按钮。
网站右上角显示自己的用户名，在repository name框内输入：用户名.github.io。这个网址将来就作为你个人博客的地址。
例如：我的用户名是zsqbigbig，那么我在repository的框内输入的为：zsqbigbig.github.io。这个地址是唯一指定的，不要乱改！如下图所示：

下面的内容不需要改动，点击Create repository按钮提交即可！

3.Git

根据自己的操作系统，下载Git。这里是Git的官网 (https://git-scm.com/downloads) 。

4.Nodejs

Nodejs的下载地址 (https://nodejs.org/en/) 点击LTS，根据自己的操作系统选择下载文件。

2.安装Hexo

1.cnpm

直接使用npm，下载安装的速度会特别慢。因此，就需要利用npm去安装良心阿里的cnpm。

打开Git Bash，输入npm install -g cnpm --registry=http://registry.npm.taobao.org。等待安装。

2.安装Hexo

打开Git Bash，输入cnpm install -g hexo-cli。安装Hexo博客框架。

可以在输入hexo -v，查看hexo安装情况。
