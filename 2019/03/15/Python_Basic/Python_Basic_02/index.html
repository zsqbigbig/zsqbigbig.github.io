2.字符串操作符

x+y       链接两字符串
n*x       将字符串x复制n次
x in y    如果x是y的子串则返回true，否则返回false

3.字符串的方法

s.lower()将全部大写字母转化为小写
s.upper()将全部小写字母转化为大写
s.split(sep=None[, num=-1])通过指定分隔符对字符串进行切片，如果参数num有指定值，则分隔num+1个子字符串。num默认为-1，即分割所有。
s.lstrip()截掉字符串左边的空格或指定字符。
s.count(str)字串str在s中出现的次数
s.replace(str1, str2)将字符串中的str1全部替换为str2 s.center(width[, fillchar])将字符串居中。width为总宽度，fillchar为填充的字符s.strip(chars)去掉字符串s左右两侧的，chars中列出的所有字符iter.join(s)将字符串iter添加到字符串s中的每两个字符之间`
