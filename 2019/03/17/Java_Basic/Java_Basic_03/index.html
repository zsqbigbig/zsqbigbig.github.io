【Java】03数据类型及运算符

发表于 2019-03-17 | 更新于 2019-03-19 | 评论数： | 阅读量：

数据类型有：int,float, double, boolean, String, Data等很多种。
运算符有：+，-，*，/,++,--,<,>……等很多种。

数据类型

基本数据类型

创建变量直接进驻内存，直接有内存地址，有默认值，小写开头

  • int
  • double
  • boolean

引用数据类型

利用new，新建了之后才进驻内存，new之前无地址，无默认值，会出现null，大写开头

  • String
  • Data
  • 数组

数据类型转换

xxx型转换为String类型

使用String.valueOf(xxx)


输出：10

String类型转换为xxx型

使用Xxx.valueOf(String)


输出：11.23

自动转换

int和double类型进行运算，结果为double类型


输出：3.3333333333333335

强制转换

将高精度转换为低精度，会丢失精度


输出：3

运算符

赋值运算符

  • =    虽然是等号，但起作用是赋值。

输出：5

一元运算符

  • + 正号
  • - 负号

算术运算符

  • +   加
  • -   减
  • *   乘
  • /   除
  • %   取余

算术赋值运算符

  • +=   
  • -=   
  • *=   
  • /=   
  • %=   

b+=a和b=b+a区别？
前者效率高，后者效率低。
后者，b+a在计算时，需要重新开辟空间。

关系运算符

  • >   大于
  • <   小于
  • ==   等于
  • <=   小于等于
  • >=   大于等于
    其返回结果为Boolean型(True/False)

自增自减运算符

  • ++    自增1
  • --    自减1
    a++:先调用a，再+1
    ++a:先+1，在调用a

逻辑运算符

  • &&    且
  • ||    或
  • !    非
    返回结果为Boolean型

小括号运算符

()
运算优先级最高，一般用来改变运算顺序

小结

上述1~8，8中运算符，优先级越来越高
*、/高于+、-
!高于&&、||
技巧：多加小括号，保证运算顺序


输出：True

    模仿。即使不理解，也要去使用。
  • 思想上：
    不难。多练习。
  • 高级阶段：
    创新=模仿+改进。
