流程控制有两种结构：分支结构和循环结构。

流程控制

选择结构

  • if
  • if-else
  • if-else-if
  • switch-case(使用很少)

补充：三目运算符
（条件?a:b)。例：

循环结构

  • while    (先判断，后循环)
  • do-while    (先循环，后判断。即使不符合条件，也会循环至少一次)
  • for    

    循环四要素：

    1. 初始化循环变量
    2. 循环条件
    3. 循环体
    4. 修改循环变量

for语法格式：
for(初始化循环变量; 循环条件; 修改循环变量){
    循环体;
}

终端循环

  • break,跳出整个循环，循环结束
  • continue,跳出本轮循环，继续执行下一次循环
