【Java】05方法（函数）

发表于 2019-03-17 | 更新于 2019-03-19 | 评论数： | 阅读量：

方法的类型及定义。方法分为有返回类型的方法和构造方法。

方法（函数）

类

对事物的抽象表示

属性（成员变量）：描述一些状态值，静态的
方法：描述一些行为，动态的

对象

类的具体化
使用对象.属性/方法 调用属性或方法

方法

语法格式：
    可访问性 返回值类型 方法名（参数）{
        函数体；
    }
目的：代码得到公共，使用时调用方法即可。
方法有如下四种情况：

无返无参

有返无参

无返有参

有返有参

return 关键字

  • 返回值
  • 结束方法

static(静态)

默认是动态：内存地址会变。
static：内存地址不变。
例如：main方法。main方法即时程序的入口，又是程序的出口。通过main方法的内存地址，进入main方法。在推出卖弄方法时，要从内存地址推出。若main方法是动态的，则main方法无法退出。

静态方法可直接调用，不需要用new新建对象调用其中的方法。
类，属性也能使用static关键字。
注意：一旦设定成static，内存只有在程序停止时才会释放资源。经常被调用的资源，常被设置为static。

方法的重载

重载（overload）
重载就是一个现象。在同一个类中，方法名相同，但是方法的参数个数和类型不同，与返回值无关。

构造方法

构造方法：在构造对象时，调用的方法。（用new新建对象时，调用的方法）
特点：与类同名，无返回值。


注意：JDK会自动为每一个类创建一个空的构造方法。但是，一点自定义构造方法，JDK将不创建。
