修改struts.xml配置文件

index.jsp索引页面。点击submit，页面跳转action=”bChose_ChoseA”。
根据struts.xml配置文件中的*_*可知，其中{1}=”bChose”,{2}=”ChoseA”。



由下面的图可知，action=”bChose_ChoseA”。ChoseA的返回值为ERROR。根据struts.xml的配置文件可知，跳转到的页面为te.jsp。

通配符 说明
* 匹配0个或多个字符除了“/”
** 匹配0个或多个字符包含“/”
\character 转义字符，“\\”匹配“\”;“\*”匹配“*”
